# Dokumentasi Penelitian - CaseMed

Direktori ini berisi dokumentasi penelitian dan materi akademik terkait kesehatan masyarakat dan lingkungan.

## Daftar Dokumen

### 1. PENELITIAN_POLUSI_UDARA_DAN_KESEHATAN_PERNAPASAN.md

**Judul Lengkap**: Penelitian Polusi Udara dan Kesehatan Pernapasan: Analisis Faktor Risiko Penyakit ISPA dengan Fokus pada Penggunaan Obat Nyamuk Semprot

**Deskripsi**: Dokumen penelitian komprehensif yang menganalisis hubungan antara berbagai faktor risiko lingkungan dan perilaku dengan kejadian Infeksi Saluran Pernapasan Akut (ISPA).

**Variabel Penelitian**:
- PM10 (Partikulat Matter)
- Jarak rumah ke sumber pencemar
- Kepadatan hunian
- Ventilasi rumah
- Perilaku merokok
- **Penggunaan obat nyamuk semprot** (Novelty)

**Isi Dokumen**:
1. **Gap Penelitian Terdahulu** - Tinjauan literatur dan identifikasi gap penelitian dengan fokus pada novelty penggunaan obat nyamuk semprot sebagai faktor risiko yang belum banyak diteliti
2. **Kemungkinan Jika Tidak Diatasi** - Analisis dampak kesehatan individu, ekonomi, sosial, dan lingkungan jika faktor risiko tidak ditangani
3. **Pendekatan Ilmu** - Kerangka teori (Ecological Model of Health), kerangka konseptual, desain penelitian (case-control study), hipotesis, dan metode analisis statistik
4. **Uraian Permasalahan dan Tujuan** - Latar belakang, rumusan masalah, tujuan penelitian (umum dan khusus), manfaat penelitian, dan keaslian penelitian

**Referensi**: Dokumen ini menggunakan 33 referensi valid dari:
- Jurnal internasional bereputasi tinggi (*The Lancet*, *New England Journal of Medicine*, *Environmental Health Perspectives*)
- Jurnal nasional Indonesia terakreditasi
- Dokumen resmi (Kemenkes RI, WHO, World Bank)
- Buku teks standar epidemiologi dan kesehatan masyarakat

**Panjang**: ~3-7 halaman (332 baris, ~3.064 kata)

**Format**: Markdown dengan struktur akademik formal

**Penggunaan**: Dokumen ini dapat digunakan sebagai:
- Outline proposal penelitian
- Referensi untuk penulisan latar belakang penelitian
- Materi pembelajaran tentang metodologi penelitian kesehatan masyarakat
- Basis untuk pengembangan protokol penelitian lengkap

2. Manfaatkan data-data epidemiologi untuk program intervensi
